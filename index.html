LaTeX Maven Plugin

See usage page.

Using this plugin you can design your graphics in LaTeX and convert them to PNG images during project build cycle. This is how the picture at this page is generated: http://www.rexsl.com/. The source of this picture is here: layers.tex.

Isn't it elegant? :)

This plugin currently not able to run on systems running any kind of Windows operating systems!
